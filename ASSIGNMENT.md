# Take-Home Challenge: Feature Usage Metrics Service

**Estimated Time:** 3-4 hours
**Difficulty:** Senior Backend Engineer

## About Unique

Unique provides AI-powered solutions for the banking industry, helping financial institutions deliver intelligent customer experiences. Our platform serves a diverse range of customers, including banks that operate in highly regulated environments with strict security requirements.

**Key Context for This Challenge:**
- Many of our banking customers run on-premises deployments without internet access
- Solutions must be self-contained and work in air-gapped environments
- We prioritize reliability, data integrity, and performance at scale
- Our systems handle time-series data for analytics and reporting

This challenge reflects real-world scenarios we encounter at Unique, where you'll build a metrics tracking system that could run in isolated banking environments.

---

## The Challenge

Build a microservice that tracks feature usage events in a chat application and computes **Daily Active Users (DAU)** metrics. The system should use scheduled aggregation to process raw events into actionable metrics.

For complete requirements and technical details, see the [README.md](./README.md) file.
